# 桌宠 Mac · 设计规范

## 1. 项目概述

**项目名称：** DeskPet  
**类型：** macOS 桌宠应用（Electron 打包）  
**核心功能：** 一个悬浮在桌面上的互动桌宠，兼具观赏性、功能性与趣味性。  
**目标用户：** macOS 用户，想要桌面增添趣味或辅助日常提醒的用户。

---

## 2. 视觉与渲染规范

### 窗口模型
- **单透明窗口**：无边框、始终置顶、点击穿透（可配置）、可拖拽
- 窗口尺寸：300×300 px（可按比例缩放）
- 抗锯齿，GPU 加速渲染

### 资源准备（用户提供的图片）
- 素材：桌宠主图（PNG，1514×1583，含透明）
- 拆分状态帧（程序化切分）：
  - 待机帧（idle）
  - 左移帧（walk_left）
  - 右移帧（walk_right）
  - 开心帧（happy）
  - 睡觉帧（sleeping）
  - 工作帧（working）
  - 说话帧（speaking）

### 动画系统
- 帧动画：CSS sprite sheet 动画，60fps requestAnimationFrame
- 状态机切换：idle → walk → idle（自由移动）| 状态变化触发表情切换
- 缓动：ease-in-out，停留时有微小上下浮动（呼吸感）

## 3. 功能模块

### 3.1 桌宠移动
- **自由漂移**：桌宠每 5-8 秒随机方向移动一次，带惯性缓动
- **边界吸附**：碰到屏幕边缘时反弹，减速停止
- **拖拽交互**：鼠标按住可自由拖拽到任意位置，松开后继续漂移

### 3.2 状态系统
多状态机，触发条件：
| 状态 | 触发条件 | 动画 |
|------|----------|------|
| idle | 默认 | 轻微上下浮动 + 眨眼 |
| happy | 点击 / 收到夸奖 | 跳一下 + 笑 |
| sleeping | 系统夜间（22:00-07:00）或手动切换 | 趴着 + "Z z z"气泡 |
| working | 手动切换（快捷键） | 端正姿势 |
| speaking | 台词气泡时 | 张嘴动画 |

状态切换：通过菜单或快捷键

### 3.3 随机台词气泡
- 气泡样式：圆角白底 + 三角指向，macOS 风格
- 触发频率：每 10-20 分钟随机一条
- 台词库（可扩展 JSON）：
  ```
  工作类：["写代码中，勿扰~", "需求还没改完吗？"]
  日常类：["今天也要加油哦！", "休息一下，喝杯水吧~"]
  时间感知：["早上好！☀️", "该睡觉啦，晚安~"]
  吐槽类：["你的代码居然跑通了！", "这个 bug 我故意的 😏"]
  ```

### 3.4 提醒事项（桌面通知）
- 内置提醒模板：
  - 💧 喝水提醒（每 45 分钟）
  - 🚶 久坐提醒（每 60 分钟）
  - 🌟 眼保健操（每 90 分钟）
  - 🍅 番茄钟（25 分钟专注 + 5 分钟休息）
- 提醒时：桌宠切换到 speaking 状态 + 气泡弹出
- 全部通过 macOS 原生通知（Notification Center）

### 3.5 右键菜单
- 显示/隐藏桌宠
- 切换状态（开心/工作/睡觉）
- 设置提醒开关
- 打开配置窗口
- 退出

### 3.6 配置窗口
- Electron 子窗口，非透明
- 功能：台词自定义、提醒间隔设置、开机自启开关、缩放比例

### 3.7 开机自启
- macOS 登录项注册（通过 AppleScript 或 launchd）

## 4. 技术方案

- **框架：** Electron（最新稳定版）
- **前端：** HTML5 Canvas + CSS 动画
- **构建工具：** electron-builder（打出 .dmg）
- **语言：** JavaScript（主）+ JSON 配置
- **图片处理：** 使用 PNG 透明帧，通过 Canvas 切分
